Jardinópolis - SP é uma das centenas de cidades integradas à API da Focus NFe. Nossos clientes já podem fazer a emissão, consulta e cancelamento de notas fiscais de serviço (NFSe). Utilizamos um padrão simples e consistente de comunicação com o web service da prefeitura, garantido disponibilidade e rapidez. Conheça o formato único que nossa API utiliza na comunicação com todos os municípios clicando aqui.

Veja abaixo os detalhes deste município:

Dados básicos:

Provedor: GissOnline.
Utiliza padrão ABRASF: Sim (2.04).
Necessita certificado digital: Sim.
Endereço produção: https://jardinopolis.giss.com.br/
Endereço homologação: Somente por Webservices
Campos em nossa API:

item_lista_servico: Utiliza o padrão nacional da Lei Complementar 116/2003.
codigo_cnae: Campo não utilizado.
codigo_tributario_municipio: Campo obrigatório, utiliza um código próprio. Consultar no XML de uma nota autorizada o valor desse campo.
Observações:

-  Município não permite o envio de "Outras retenções"
