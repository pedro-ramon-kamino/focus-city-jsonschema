Erechim - RS é uma das centenas de cidades integradas à API da Focus NFe. Nossos clientes já podem fazer a emissão, consulta e cancelamento de notas fiscais de serviço (NFSe). Utilizamos um padrão simples e consistente de comunicação com o web service da prefeitura, garantido disponibilidade e rapidez. Conheça o formato único que nossa API utiliza na comunicação com todos os municípios clicando aqui.

Veja abaixo os detalhes deste município:

Dados básicos:

Provedor: System Pro.
Utiliza padrão ABRASF: Sim (2.01).
Necessita certificado digital: Sim.
Endereço produção: https://www.nfse.erechim.rs.gov.br:8181/
Endereço homologação: Somente por Webservices.
Campos em nossa API:

item_lista_servico: Utiliza o padrão nacional da Lei Complementar 116/2003.
codigo_cnae: Campo não utilizado.
codigo_tributario_municipio: Campo não utilizado.
Observações:

-  Para emissão de notas via webservices é necessário solicitar liberação junto à prefeitura. Esse processo é realizado seguindo o passo a passo abaixo:
   Portal da prefeitura (não é necessario logar) > Solicitação de uso da NFS-e, onde primeiramente deverá inserir o CNPJ e selecionar a opção: Utiliza sistema próprio com acesso a partir de WebServices ( Iniciar a 1a. Fase de Homologação ) Em seguida inserir a senha de acesso ao portal a data atual, e enviar. 
Logo após deverá realizar o mesmo procedimento porem selecionando a opção: Utiliza sistema próprio com acesso a partir de WebServices ( Liberação para Produção, após efetuada a 1a. Fase de Homologação )
