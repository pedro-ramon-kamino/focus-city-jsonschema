Duque de Caxias - RJ é uma das centenas de cidades integradas à API da Focus NFe. Nossos clientes já podem fazer a emissão, consulta e cancelamento de notas fiscais de serviço (NFSe). Utilizamos um padrão simples e consistente de comunicação com o web service da prefeitura, garantido disponibilidade e rapidez. Conheça o formato único que nossa API utiliza na comunicação com todos os municípios clicando aqui.

Veja abaixo os detalhes deste município:

Dados básicos:

Provedor: IssNet (v.B)
Utiliza padrão ABRASF: Sim (2.04).
Necessita certificado digital: Sim.
Endereço produção: https://issnetonline.com.br/duquedecaxias/
Endereço homologação: https://www.issnetonline.com.br/homologaabrasf/
Campos em nossa API:

item_lista_servico: Utiliza o padrão nacional da Lei Complementar 116/2003.
codigo_cnae: Campo obrigatório.
codigo_tributario_municipio: Campo obrigatório, geralmente utiliza o valor do item_lista_servico sem pontuação (ex.: para 1.05 > 105). Consultar no XML de uma nota autorizada o valor desse campo.
Observações:

- Para emitir notas no ambiente de produção via webservice, deve-se anteriormente, solicitar autorização para emissão, enviando os dados da empresa prestadora para suporte@notacontrol.com.br
    - Caso mesmo solicitando a liberação não seja possível emitir, com a validação:
” A empresa não está habilitada no ambiente de integração”
    - Será necessário alterar o plano de emissão de notas.
    - Para maiores informações basta solicitar nesse mesmo e-mail;
- Após liberarem para emissão via webservice, deverá liberar o RPS. Basta acessar https://www.issnetonline.com.br/duquedecaxias/online/login/Login.aspx, logar com usuário e senha do cliente, ir em Solicitação, selecionar o modelo do Documento Fiscal (no caso RPS – Recibo Provisório de Serviço) e a quantidade de notas a serem emitidas que desejará deixar configurado.
