Caçapava - SP é uma das centenas de cidades integradas à API da Focus NFe. Nossos clientes já podem fazer a emissão, consulta e cancelamento de notas fiscais de serviço (NFSe). Utilizamos um padrão simples e consistente de comunicação com o web service da prefeitura, garantido disponibilidade e rapidez. Conheça o formato único que nossa API utiliza na comunicação com todos os municípios clicando aqui.

Veja abaixo os detalhes deste município:

Dados básicos:

Provedor: Etransparencia.
Utiliza padrão ABRASF: Não.
Necessita certificado digital: Não. Requer par de tokens exclusivos para webservice.
Endereço produção: https://nfe.etransparencia.com.br/sp.cacapava/nfe/principal.aspx
Endereço homologação: https://nfehomo.etransparencia.com.br/sp.cacapava/nfe/principal.aspx
Campos em nossa API:

item_lista_servico: Utiliza o padrão nacional da Lei Complementar 116/2003.
codigo_cnae: Campo não utilizado.
codigo_tributario_municipio: Campo não utilizado.
Observações:

- Obrigatório informar os campos valor_iss e aliquota.

- A autenticação do webservice é feita usando um par de tokens disponíveis nos cadastros do portal da prefeitura: Código do Usuário cadastrado; Código da empresa Contribuinte .

- Os códigos são diferentes do login e senha de acesso ao portal.
Para o Login: Sistema > Segurança > Editar Perfil > Código do usuário para login em Webservices
- Para a senha: Contribuintes > Cadastro de Contribuintes > Consultar > Código (abaixo de Razão Social).
