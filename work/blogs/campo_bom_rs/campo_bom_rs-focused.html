Já é possível emitir NFSe para o município de Campo Bom/RS usando o nosso formato único de integração.

Visite a documentação para descobrir como integrar a emissão de NFSe a partir do seu sistema.

O município de Campo Bom/RS usa o provedor Infisc que segue um padrão próprio e possui um ambiente de testes (homologação) cujo cadastro pode ser feito por e-mail.

Confira aqui a nossa lista com todas as cidades atendidas.

Provedor: Infisc. Utiliza padrão ABRASF: Não. Necessita certificado digital: Sim. Endereço produção: https://nfse.campobom.rs.gov.br Endereço homologação: https://campobom-homol.infisc.com.br Processo para liberação do ambiente de homologação: Envie e-mail para o suporte informando o CNPJ, razão social e o regime tributário da empresa (optante pelo Simples Nacional ou regime normal). item_lista_servico: Utiliza um padrão próprio. Consulte uma nota já emitida o campo "código de serviço" ou entre na interface de emissão manual da prefeitura para verificar os códigos disponíveis. codigo_cnae: Não é utilizado. codigo_tributario_municipio: Não é utilizado.

Observações

Numeração e série das notas: Ao contrário das demais cidades, a numeração e série da NFSe não é controlada pela prefeitura e sim pela nossa API. Será necessário que você informe o número e série da última nota já emitida para o nosso suporte para que você possa emitir as notas pela nossa API.

Campos adicionais: Esta prefeitura contém diversos campos adicionais além dos que o padrão da API disponibiliza. No momento é possível utilizar apenas os campos disponíveis no nosso padrão único que a API utiliza. Caso precise utilizar algum campo adicional, entre em contato com o suporte.
