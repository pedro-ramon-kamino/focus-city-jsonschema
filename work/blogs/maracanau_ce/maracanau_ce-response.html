Como emitir NFSe em Maracanaú / CE - API Focus NFe

Maracanaú / CE é uma das centenas de cidades integradas à API da Focus NFe. Veja abaixo os detalhes deste município e conheça a nossa API com formato único de comunicação para todos os municípios clicando aqui.

Provedor: SpeedGov (Intersol).

Utiliza padrão ABRASF: Sim, versão 1..

Necessita certificado digital: Sim

Endereço produção:http://servicos2.speedgov.com.br/maracanau Endereço homologação:http://speedgov.com.br/satmodws/servlet/com.satweb.hnsefin

Campos dos itens:

item_lista_servico: Utiliza o padrão nacional da Lei Complementar 116/2003. Só é permitido utilizar os códigos que foram informados no momento do cadastro da empresa.

codigo_cnae: Não é utilizado.

codigo_tributario_municipio: Não é utilizado.

É necessário fazer cadastro em homologação. Link de Cadastro http://speedgov.com.br/satmodws/servlet/com.satweb.hnsefin Link de Homologação http://speedgov.com.br/wsmod/Nfes?wsdl Link para Visualizar erros e corrigir http://speedgov.com.br/satmodws/servlet/com.satweb.hnsefin

No cadastro em homologação é gerada uma Inscrição Municipal diferente, que deve ser enviada ao suporte junto com o CNPJ da empresa, para liberação do ambiente. Nos envios em homologação, a empresa precisará alterar a Inscrição Municipal no cadastro para poder enviar.
