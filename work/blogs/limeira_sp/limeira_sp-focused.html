Limeira - SP é uma das centenas de cidades integradas à API da Focus NFe. Nossos clientes já podem fazer a emissão, consulta e cancelamento de notas fiscais de serviço (NFSe). Utilizamos um padrão simples e consistente de comunicação com o web service da prefeitura, garantido disponibilidade e rapidez. Conheça o formato único que nossa API utiliza na comunicação com todos os municípios clicando aqui.

Veja abaixo os detalhes deste município:

Dados básicos:

Provedor: Iibrasil.
Utiliza padrão ABRASF: Sim (2.0).
Necessita certificado digital: Sim, Mas possui algumas particularidades. Também utiliza um token que precisa ser informado como senha.
Endereço produção: https://limeira.iibr.com.br/login.php
Endereço homologação: https://limeira.iibr.com.br/login.php
Campos em nossa API:

item_lista_servico: Utiliza o padrão nacional da Lei Complementar 116/2003.
codigo_cnae: Campo não utilizado.
codigo_tributario_municipio: Campo não utilizado.
Observações:

- Cancelamento de notas somente pelo portal da prefeitura;
- A Série do RPS é alfanumérica, geralmente é informada como "RPSL".

- O token é gerado no site da prefeitura. Validade de 6 meses:
SEGISS > RPS > Tokens > Criar Token
