Carapicuíba - SP é uma das centenas de cidades integradas à API da Focus NFe. Nossos clientes já podem fazer a emissão, consulta e cancelamento de notas fiscais de serviço (NFSe). Utilizamos um padrão simples e consistente de comunicação com o web service da prefeitura, garantido disponibilidade e rapidez. Conheça o formato único que nossa API utiliza na comunicação com todos os municípios clicando aqui.

Veja abaixo os detalhes deste município:

Dados básicos:

Provedor: Giap.
Utiliza padrão ABRASF: Não.
Necessita certificado digital: Não, utiliza token gerado no acesso da prefeitura.
Endereço produção: https://carapicuiba.giap.com.br/apex/carapi/f?p=1386:101
Endereço homologação: Somente por Webservices (não é um ambiente de verdade, apenas simula uma emissão).
Campos em nossa API:

item_lista_servico: Utiliza o padrão nacional da Lei Complementar 116/2003.
codigo_cnae: Campo não utilizado.
codigo_tributario_municipio: Campo não utilizado.
Observações:

- O token para acesso deve ser gerado no portal da prefeitura:
    - Acessar o portal da prefeitura > menu Dados Cadastrais.
